# Змейка (Vanilla JS + Canvas)

Простая игра «Змейка», работает в любом современном браузере. Управление — стрелки или WASD, пауза — Пробел, перезапуск — R. На тач-экранах доступны кнопки.

## Запуск локально

Вариант 1: открыть файл напрямую
- Откройте `index.html` в браузере (перетащите файл на окно браузера).

Вариант 2: через локальный сервер (рекомендовано)
```bash
python3 -m http.server 8080 --directory /workspace/snake
```
Затем откройте `http://localhost:8080` в браузере и перейдите к `index.html`.

## Структура
- `index.html` — разметка и HUD
- `style.css` — оформление
- `script.js` — логика игры (рендер, столкновения, еда, скорость, рекорд)

## Возможности
- Пауза/продолжение (Пробел)
- Перезапуск (R)
- Накапливаемый рекорд (localStorage)
- Адаптивное поле и кнопки управления для мобильных
