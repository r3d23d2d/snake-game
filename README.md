# Автоматическая минусовка для Яндекс Директа

Простой сайт, который использует ChatGPT для генерации минус-слов на основе списка фраз.

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Задайте переменную окружения `OPENAI_API_KEY` со своим ключом OpenAI.
3. Запустите сервер:
   ```bash
   node server.js
   ```
4. Откройте [http://localhost:3000](http://localhost:3000) в браузере и введите фразы.

## Структура

- `server.js` — Express-сервер с эндпоинтом `/api/minus`.
- `public/` — статические файлы фронтенда.
